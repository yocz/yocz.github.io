Gitを初めてみたよ

Gitを初めてみたので、初歩の初歩の簡単なメモ。

###リポジトリの初期化

###リポジトリの状態を確認する

###ステージ領域にファイルを追加する

ディレクトリ内のすべてのファイルを対象にする場合

###リポジトリのcommit

もしくは、

-mを省略すると、エディタが起動する
1行目にコメント
2行目は空白
3行目以降はcommitの詳細を記述する

###commitのステータスの確認

オプション

  • -pretty=short : logの1行目のみ表示
  • git log : 入力したディレクトリ、ファイルに関するlogを表示
  • -p : 差分を表示

##差分を表示

commitの内容と編集した内容のdiffを表示する
commitする前にgit diff してからcommitする癖つけとくとよいみたい
