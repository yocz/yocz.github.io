Gitを初めてみたよ

Gitを初めてみたので、初歩の初歩の簡単なメモ。

###リポジトリの初期化

###リポジトリの状態を確認する

###ステージ領域にファイルを追加する

ディレクトリ内のすべてのファイルを対象にする場合

###リポジトリのcommit

もしくは、

-mを省略すると、エディタが起動する
1行目にコメント
2行目は空白
3行目以降はcommitの詳細を記述する

###commitのステータスの確認

オプション

  • -pretty=short : logの1行目のみ表示
  • git log : 入力したディレクトリ、ファイルに関するlogを表示
  • -p : 差分を表示

##差分を表示

commitの内容と編集した内容のdiffを表示する
commitする前にgit diff してからcommitする癖つけとくとよいみたい

ブログ初めてみたよ

俺、25歳、初めてブログを初めてみようと思った。
ブログを書く環境としては、アメブロ、はてな、wordpress辺りが主流なのかな、と思ってみたけど、ちょっと挑戦してみたいなあと思って、調べてみると、githubpagesっていうものがあって、なんと無料で使えるらしい。
無料という言葉に引かれて、早速環境を構築してみた。
今回はサイトジェネレーターとしてHEXOを使用した。
HEXOとは、node.jsで作成された静的ページジェネレータ、らしい。

##構築の手順
LIGのサイトを参考にした。
nodeもHEXOも全くの初めてだったけどなんとかできたよ。
レッツトライ。
1.githubページを用意する
http://liginc.co.jp/web/html-css/html/96453

2.node.jsの環境を構築する
http://liginc.co.jp/web/programming/node-js/85318

3.hexo導入
http://liginc.co.jp/web/programming/server/104594

ドメインはムームードメインで取得して、DNSサービスはGehirn DNSを利用した。
画像はFlickrにおく予定。1TB無料ってすごいよね。
テーマはapolloを使用した。

##HEXOを導入してみての感想
vimで記事の作成から投稿までできるので、なんかかっこいい。便利な使い方とかカスタマイズとかいろいろあるみたいなので、これからいろいろ試してみたい。
